# Interactive-Cat-Toy
UCF Senior Design Project ( Fall 2021 - Spring 2022 )

#---- Material programmed with code: ----#
    
    WS2812 LED strip
    ServoMotor
    STEMMA Speaker
    HC-SR501 Sensor 
    HC-05 Bluetooth

#---- Application Signals: ----#

1st digit is function 
      0 - lighting
      1 - sound
      2 - tail motion
2nd digit is pattern / sound
      0 - Option 1 (base)
      1 - Option 2
      2 - Option 3
      3 - Option 4 (off)
Ex: O1 changes lighting to pattern option 2
